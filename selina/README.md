# **_PROJECT CONFIGURATION GUIDE (LOCAL)_**
## Runtime environment
#### &nbsp; &nbsp; Selina requires [Node.js](https://nodejs.org/en/download/) v10+ to run.
## Install the Dependencies, DevDependencies and run all the Services
```
cd <path/to/source>
npm i
npm start
```
